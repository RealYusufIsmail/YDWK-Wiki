# How to install

To install YDWK, add the following to your `build.gradle.kts`:

```kotlin
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.github.realyusufismail:ydwk:${project.version}")
}
```

project.version is the latest version of YDWK. You can find the latest
version [here](https://maven-badges.herokuapp.com/maven-central/io.github.realyusufismail/ydwk/badge.svg)](https://maven-badges.herokuapp.com/maven-central/io.github.realyusufismail/ydwk)
